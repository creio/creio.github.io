{{- $isStaticFallback := false -}} {{- $ctx := .context -}} {{- /* 1. Читаем параметры из конфига [site_images.defaults] */}} {{- $defaults := site.Params.site_images.defaults -}} {{- /* Читаем качество из конфига, если его там нет — ставим дефолт */}} {{- $quality := $defaults.quality | default 80 -}} {{- /* Настройка кропа: из вызова -> из конфига -> дефолт */}} {{- $processRule := .process | default $defaults.process | default "fill 1280x720 Center" -}} {{- $image := "" -}} {{- /* Сначала проверяем, передали ли src напрямую в partial, если нет — берем из Front Matter страницы */}} {{- $rawSrc := .src | default $ctx.Params.image | default "" -}} {{- $altText := .alt | default $ctx.Title -}} {{- $customClass := .class | default "" -}} {{- $addClass := printf "%s %s" ($defaults.imageAddClass | default "img-fluid lazyload blur-up") $customClass | strings.TrimSpace -}} {{- if and (not $rawSrc) $ctx.Params.images -}} {{- $rawSrc = index $ctx.Params.images 0 -}} {{- end -}} {{- if $rawSrc -}} {{- /* Безопасно очищаем путь от ведущих слэшей и точек */}} {{- $cleanSrc := strings.TrimPrefix "/" $rawSrc -}} {{- $cleanSrc = strings.TrimPrefix "./" $cleanSrc -}} {{- /* ШАГ 1: Ищем как ресурс внутри папки с постом через $ctx (Page Bundle) */}} {{- $image = $ctx.Resources.GetMatch $cleanSrc -}} {{- /* ШАГ 2: Если не нашли, ищем глобально в assets/ */}} {{- if not $image -}} {{- $assetsSrc := strings.TrimPrefix "assets/" $cleanSrc -}} {{- $image = resources.Get $assetsSrc -}} {{- end -}} {{- /* ШАГ 3: Если всё ещё не нашли, проверяем, не указывает ли путь на static/images */}} {{- if not $image -}} {{- /* Убираем префикс static/, если пользователь случайно указал его вручную */}} {{- $staticClean := strings.TrimPrefix "static/" $cleanSrc -}} {{- /* Проверяем физическое существование файла в папке static/ */}} {{- if fileExists (printf "static/%s" $staticClean) -}} {{- $isStaticFallback = true -}} {{- /* Формируем прямую ссылку на static-файл для вывода на сайт */}} {{- $rawSrc = printf "/%s" $staticClean -}} {{- end -}} {{- end -}} {{- end -}} {{- /* ШАГ 4: Если файл вообще нигде не найден, берем заглушку из assets/images/ */}} {{- if and (not $image) (not $isStaticFallback) -}} {{- $defaultImg := $defaults.defaultImage | default "default-image.png" -}} {{- $image = resources.Get (printf "images/%s" $defaultImg) -}} {{- end -}} {{- /* Умный расчет адаптивных sizes на основе processRule */}} {{- $sizesAttr := "" -}} {{- if eq $defaults.sizes "auto" -}} {{- $sizesAttr = "auto" -}} {{- else if $defaults.sizes -}} {{- $sizesAttr = $defaults.sizes -}} {{- else -}} {{- $widthNum := replace (replace (replace $processRule "resize " "") "fill " "") "x" "" -}} {{- /* Отсекаем высоту и параметры кропа, если передано например "1280x720 Center" */}} {{- $widthNum = index (split $widthNum " ") 0 -}} {{- $widthNum = index (split $widthNum "x") 0 -}} {{- $sizesAttr = printf "(max-width: %spx) 100vw, %spx" $widthNum $widthNum -}} {{- end -}} {{- /* --- РЕНДЕРИНГ HTML --- */}} {{- if $isStaticFallback -}} {{- /* ВАРИАНТ А: Картинка из /static/. Выводим как обычный тег, так как ресайз невозможен */}} {{- else if $image -}} {{- /* ВАРИАНТ Б: Картинка обработана через ресурсы (Page Bundle или Assets) */}} {{- /* Применяем кроп/fill из параметров */}} {{- $processedImage := $image.Fill $processRule -}} {{- /* Собираем адаптивный srcset из ширин вашего конфига с качеством $quality */}} {{- $imgSrcSet := slice -}} {{- $widths := $defaults.widths | default (slice 360 480 640 768 940 1024 1366) -}} {{- range $widths -}} {{- if le . $image.Width -}} {{- $webpUrl := (printf "%dx webp q%d" . $quality | $image.Resize).RelPermalink -}} {{- $imgSrcSet = $imgSrcSet | append (printf "%s %dw" $webpUrl .) -}} {{- end -}} {{- end -}} {{- $imgSrcSet = (delimit $imgSrcSet ", ") -}} {{- /* БЕЗОПАСНАЯ ГЕНЕРАЦИЯ LQIP (микро-превью для эффекта размытия) */}} {{- $lqipRule := $defaults.lqip | default "16x webp q20" -}} {{- /* На случай, если в конфиге передали просто число, приводим к формату */}} {{- if not (in (string $lqipRule) " ") -}} {{- $lqipRule = printf "%sx webp q20" (strings.TrimSuffix "x" (string $lqipRule)) -}} {{- end -}} {{- $lqip := $image.Resize (string $lqipRule) -}} {{- /* Дефолтный src (наш пожатый webp) */}} {{- $finalSrc := (printf "%dx webp q%d" $processedImage.Width $quality | $processedImage.Resize).RelPermalink -}} {{- $smallLimit := int (site.Params.smallLimit | default 300) -}} {{- if gt $processedImage.Width $smallLimit -}} {{- /* Для больших картинок используем ленивую загрузку с LQIP через data-src */}} {{- else -}} {{- /* Для мелких иконок/картинок srcset не нужны */}} {{- end -}} {{- else -}} {{- errorf "Image layout failed: No file found for page '%s'" $ctx.File.Path -}} {{- end -}}
