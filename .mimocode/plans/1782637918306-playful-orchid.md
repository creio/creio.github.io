## 3. FlexSearch vendor — удалить 1.27MB

**Файл:** `assets/js/vendor/flexsearch/` (1.27MB)

Это замонтированный из `node_modules/flexsearch` дистрибутив. Hugo сам монтирует его через `config.toml`:

```toml
[[module.mounts]]
  source = "node_modules/flexsearch"
  target = "assets/js/vendor/flexsearch"
```

Файлы в `assets/js/vendor/flexsearch/` — это копия, которая перезаписывается при `npm ci`. Если Hugo корректно монтирует через module mounts — локальная копия не нужна.

**Действие:** Удалить `assets/js/vendor/flexsearch/`. Hugo будет читать из `node_modules` через mount.
