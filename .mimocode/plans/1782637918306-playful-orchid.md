# Plan: Fix Netlify deploy failure

## Проблема
Netlify: "Failed to install Node.js version '24.18.0'" — Netlify не поддерживает Node 24.x

## Решение

### netlify.toml
Заменить:
```toml
NODE_VERSION = "24.18.0"
NPM_VERSION = "11.12.1"
```
На:
```toml
NODE_VERSION = "22"
NPM_VERSION = "11"
```

Node 22 — текущий LTS, точно поддерживается Netlify.

### GitHub Actions warning
Warning "Node.js 20 is deprecated" — это про внутренний Node в `actions/checkout@v4` и `actions/setup-node@v4`. Это их проблема, не наша. Ничего не менять.

## Verification
- Push → Netlify deploy succeeds
