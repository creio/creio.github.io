# Personal website — hugo

## Requirements

- [Git](https://git-scm.com/) — latest source release
- [Node.js](https://nodejs.org/) — latest LTS version or newer

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run start
```
